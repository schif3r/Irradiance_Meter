Irridiance Meter : Measuring Solar Irridiance or how sunlight is effective on the location. This project demonstrates real-time measurement, smoothing, and noise handling with interactivity and feedback.
This project sits at the intersection of, Renewable Energy, Embedded Systems, Data Science, and Smart Automation.

Hardware Inventory :

ESP32  (pins 34/35 usage)

LCD 20*4(I2C interface)

Solar panel 6v (connected to analog pin 34)

LDR (Light Dependent Resistor, connected to analog pin 35)

Buzzer

TP5100

2x Battery 18650 3.7v
